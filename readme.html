Gate server side

Gate is a all-in-one malware for educational purpose only. It has been built to illustrate a journalistic video about cybersecurity. The developpement and ideas are open-source and maintained by @CallMeKitsu. The usage of this kind of programs is punished by years of jail at minus, stay away from problems and don't use it.

Dependancies

The app uses Node.js runtime and the node packages colors to style the CLI and socket.io to build the socket connection between clients and server. As the server acts as a command-line remoter for all the differents clients, no UI design / package is needed. The whole controls are included into the native Node.js console.
